## 🎨 Styles and Templates

### Understanding Styles
Styles are predefined formatting combinations applied to text for quick and consistent appearance.

**Key Benefits:**
- Consistent formatting across documents
- Easy to update multiple elements at once
- Professional look with minimal effort

**Common Style Types:**
- **Heading Styles**: For document structure (Heading 1, 2, 3)
- **Paragraph Styles**: Control paragraph formatting
- **Character Styles**: Apply to selected text only
- **Linked Styles**: Apply to paragraphs or selected text

---

### ✅ Creating Custom Styles
1. Format text as desired  
2. Select the formatted text  
3. Right-click → **Styles → Save Selection as a New Quick Style**  
4. Name your style and click **OK**

---

### Using Templates
Templates are pre-designed documents to save time.

**Built-in Templates:**
- Letters  
- Reports  
- Resumes  
- Newsletters  
- Brochures  

**Creating Custom Templates:**
1. Design your document  
2. Go to **File → Save As**  
3. Choose **Word Template (*.dotx)**  
4. Save in the Templates folder  

---

## 🖋 Headers, Footers, and Page Numbers

### Adding Headers and Footers
**Steps:**
1. Go to **Insert → Header** or **Insert → Footer**  
2. Choose a built-in design or select **Edit Header/Footer**  
3. Type your content  
4. Double-click outside to exit  

**Common Elements:**
- Document title  
- Page numbers  
- Date and time  
- Company logo  
- Author name  

---

### Page Numbers
**Automatic Page Numbering:**
1. Go to **Insert → Page Number**  
2. Choose position (top, bottom, margins)  
3. Select a design style  

**Custom Page Numbering:**
- Start from a specific number  
- Use different formats (1, 2, 3 or A, B, C)  
- Include chapter numbers  

---

### Different Headers/Footers
- **First Page Different**: Great for title pages  
- **Odd & Even Pages**: Common in books/reports  

---

## 🤝 Advanced Collaboration Features

### Track Changes
Track all modifications made to a document.

**Enable Track Changes:**
- Go to **Review → Track Changes**  
- Click to turn it on  

**Review Changes:**
- Accept or reject changes  
- Add comments for clarity  

---

### Comments
**Add Comments:**
1. Select text  
2. Go to **Review → New Comment**  

**Manage Comments:**
- Reply  
- Resolve  
- Delete  

---

### Document Comparison
Compare two versions of a document:
1. Go to **Review → Compare**  
2. Choose original and revised documents  
3. Review the differences  

---

## 📰 Professional Document Layout

### Section Breaks
Allow different formatting in different sections.

**Types:**
- **Next Page**: Starts new section on next page  
- **Continuous**: Same page  
- **Even/Odd Page**: Starts on specific page types  

---

### Columns
Create multi-column layouts:
1. Select text  
2. Go to **Layout → Columns**  
3. Choose number of columns  

---

### Table of Contents (TOC)
Automatically generate a TOC:
1. Apply heading styles  
2. Place cursor where TOC should appear  
3. Go to **References → Table of Contents**  
4. Choose a style  

**Update TOC:** Right-click → **Update Field**  

---

## ✅ Best Practices

### Document Organization
- Use consistent heading styles  
- Logical structure  
- Use page and section breaks properly  

### Collaboration
- Always use Track Changes  
- Add clear comments  
- Use meaningful file names  

### Professional Appearance
- Use templates  
- Apply styles, not manual formatting  
- Include headers, footers, and page numbers
